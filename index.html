Proyectos

01

Buscaminas

El objetivo del Buscaminas es descubrir todas las celdas seguras en un tablero sin destapar ninguna mina. Cada celda puede ocultar una mina o mostrar un número que indica cuántas minas hay en las celdas adyacentes. El jugador debe usar estos números para deducir dónde están las minas, marcarlas con banderas y seguir descubriendo celdas seguras. Si destapa una mina, el juego termina. Gana quien descubre todas las celdas seguras sin errores.

02

Tres en raya

Este juego es una versión del clásico 3 en raya con dos modalidades: de 6 y 9 fichas. En el modo de 6 fichas, cada jugador tiene 3; si colocan las tres fichas sin hacer una línea, pueden mover una de sus fichas para intentar ganar. En el modo de 9 fichas, el juego es el tradicional: cuando se llenan las casillas, el juego termina. Además, ofrece tres modos de juego: 'jugador vs aleatorio', 'jugador vs IA' y '2 jugadores', permitiendo diferentes formas de competir.

03

Tienda online

Esta es una tienda de relojes online que incluye las funcionalidades generales de cualquier e-commerce. Desarrollada con HTML, CSS, JavaScript, jQuery y PHP, permite realizar operaciones CRUD para gestionar productos y conectarse con su base de datos. Incluye un sistema de administración, donde los roles deben ser asignados manualmente desde la base de datos. El sitio ofrece una experiencia completa de compra, incluyendo gestión de usuarios, carrito, y un buscador optimizado para filtrar productos por marca o nombre.

04

Modelo predictivo

Un modelo de machine learning diseñado para predecir si una persona tiene diabetes analiza datos médicos relevantes y aprende a identificar patrones asociados con esta condición. El modelo utiliza algoritmos avanzados de aprendizaje supervisado, como Random Forest, K-Nearest Neighbors (KNN) y árboles de decisión, para evaluar con que entrenamiento podemos sacar mejor rendimiento al modelo.
